# Django

## Создание venv
```python
sudo apt install python3-venv
python3 -m venv venv
source venv/bin/activate
```
## Установка


Устанавливаем Модули:
```python
pip install django
pip install django-extensions
```

## Загружаем данные из json файла:
```python
python manage.py loaddata data.json
```
